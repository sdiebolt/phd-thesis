## Compilation

This thesis is written in [Typst](https://typst.app/).

The following fonts must be installed system-wide:

- [Fira Sans](https://github.com/mozilla/Fira) — body text
- [Fira Code](https://github.com/tonsky/FiraCode) — code blocks
- [Fira Math](https://github.com/firamath/firamath) — math equations

```bash
typst compile main.typ
```
